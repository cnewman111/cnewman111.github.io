Language Universe

Visit Repo

Work in progress. Language Universe is a SaaS web app designed to teach people of all occupations, age ranges, and walks of life foreign language in an entertaining yet practical way. Users can create detailed characters using natural language, then talk to those characters in any language they choose with instantaneous grammar feedback. By placing users in a variety of situations ranging from ordinary to extraordinary, they learn the language more fully than by exclusively learning from a textbook. The app is based on ruby on rails and deployed on fly.io.

SIXT33N: Voice Controlled Robot

SIXT33N is a voice controlled robot that peforms actions based on the voice of the user. It uses classification to only work on the voice of the person it was trained on so that it is unusuable by unauthorized users. This functionality is implemented through a microphone processing circuit on the robot itself as well as optimization models (total least squares and orthogonal distance regression) in the arduino's code. Principal component anaylsis is used to reduce the dimensionality of the model, allowing computationally intensive classification calculations to be quickly performed by the basic computer on the arduino. All of the control circuits, signal processing circuits, and power supply circuits are created from scratch.

Dijsktra's Demons - Video Game

Dijkstra’s Demons is a turn based game where the player must navigate a maze, find a key, and escape through a secret entrance all while avoiding demons that pursue the player. Usable items spawn randomly on the floor of the map, which the player can use to help escape the maze. The game generates unique player navigable maps using pseudo random numbers, so every world is different. Enemies (Demons) pursue the player through the map using the most efficient path, hence the name “Dijkstra’s Demons" named after Dijsktra's algorithm. The music was specifically created by me for the game.

UCSB Biology Lab

During this internship at a UCSB Developmental Biology Laboratory, I employed the Polymerase Chain Reaction (PCR) method to test the function of a specific gene in a small aquatic organism.

I prepared solutions of the template DNA, Taq Polymerase, Primers, and Nucleotides which I then added to a heat cycler where I performed DNA replication via PCR.

To complete the project, I prepared the solution containing the replicated DNA for gel electrophoresis and separated the modified DNA from the rest of the solution.
